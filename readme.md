# Help game for NetHack Sokoban.

## Usage

```bash
./Sokoban.exe mapfile [actionfile]
```

## Key:
h/←: move left  
j/↓: move down  
k/↑: move up  
l/→: move right  
backspace: undo last action  

You can copy actions and save to a text file as actionfile.
